# testing_claude_mobile

Testing mobile coding capabilities with Claude and Cursor!

## The Bet

This repository contains proof that both Claude and Cursor support coding from mobile devices.

## Files

- `dinner-popup.html` - A fun popup message reminding Jingxia about the dinner he owes from our bet! Open this file in any browser to see the message.

## How to View

Simply open `dinner-popup.html` in any web browser on your phone or computer to see the popup message!
